# Инструкции по сборке проекта

1. **Запуск SQL-скрипта для создания базы данных:**
   - Используйте файл `weatherdbScript.sql` для создания базы данных.

2. **Изменение строки подключения в appsettings.json:**
   - Откройте файл `appsettings.json`.
   - Найдите секцию "ConnectionStrings" и измените значение "WeatherAppCon" в соответствии с вашей базой данных.

3. **Изменение URL в файле main.js для доступа к API:**
   - Откройте файл `main.js`.
   - Найдите `apiURL` и замените ее значением на ваш URL.

4. **Запуск проекта в Visual Studio:**
   - Запустите WeatherApp в Visual Studio.

5. **Открытие веб-приложения:**
   - Откройте файл `index.html` в веб-браузере для взаимодействия с приложением.
